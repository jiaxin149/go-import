# golang库备忘录

- **Golang每日一库** https://juejin.cn/column/7289249524324466724
>golang每日一库，能发现很多优秀的库

- **ristretto** https://github.com/dgraph-io/ristretto
>本地带过期的高性能kv缓存库

- **queue** https://github.com/golang-queue/queue
>轻量的go协程管理库

- **fsnotify**  https://github.com/fsnotify/fsnotify
>跨平台的文件事件监听库

- **gin-contrib / i18n** https://github.com/gin-contrib/i18n
> gin框架多语言中间件

- **dongle** https://github.com/dromara/dongle
> 轻量化的编解码、加解密、哈希摘要库

- **sqids-go**  https://github.com/sqids/sqids-go
> 数字转唯一简短id库

- **freecache**  https://github.com/coocood/freecache
> 0 GC开销的、带过期时间的内存kv缓存库（最后更新2024年）

- **bigcache** https://github.com/allegro/bigcache
> 高性能带过期时间的内存kv库，比freecache性能更强

- **GoPay** https://github.com/go-pay/gopay
>微信、支付宝、通联支付、拉卡拉、PayPal、Apple 的Go版本SDK。【极简、易用的聚合支付SDK】

- **qqwry** https://github.com/xiaoqidun/qqwry
>基于内存的高性能纯真IP（包含v6）查询库

- **chenyahui / gin-cache** https://github.com/chenyahui/gin-cache
>比gin官方版本性能更强的的请求缓存中间件
